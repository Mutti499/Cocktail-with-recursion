Welcome to Drinking Straw with Recursion

In the "Cocktail with loop" repository: I wrote a Python program to implement the ASCII cocktail glasses below using two parameters glass_size as the size of the cocktail glass and straw_pos as the position of the straw. Purpose of my code was drawing a cocktail that decreases as you drink it through a straw. One example run is given at the bottom of this page. You can find other example runs in the example_runs.txt file.

Description

In this project, I again drew the same shape as in the "Cocktail with loop" repository. But this time, I implemented it with recursions instead of for loops. Some of the rules in my implementation:

  • I used recursions to draw the shapes.
  • I am not allowed to use any kind of loops.
  • I am not allowed to use lists or string variables.
  • I am not allowed to use multiplication in print statements.
  • I can only print one of the followings in your print statements: ' ', '*', 'o', '\\', '/', '||', '--'.
  • straw_pos is not be greater than glass_size*2 (straw_pos <= glass_size*2).
  • straw_pos and glass_size will be strictly greater than zero (straw_pos > 0 and glass_size > 0).
